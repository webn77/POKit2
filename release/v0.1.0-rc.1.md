# POKit Starter v0.1.0-rc.1

## Summary

POKit Starter is a local-first PM/PO AI Harness. It uses `.ai-os/` as the project source of truth for issues, evidence, gates, memory, and next actions.

This release candidate packages the sanitized starter bundle for first public GitHub release testing.

## What Is Included

- Starter `AGENTS.md`
- Starter `README.md`
- `pokit.config.yaml`
- MIT `LICENSE`
- Public-safe `.ai-os/` starter files
- Starter runner: `scripts/pokit-runner.mjs`
- Starter doctor: `scripts/pokit-doctor.mjs`

## Verification

- Local archive is extracted into a fresh temporary project.
- Starter runner passes with active issue `POK-001`.
- Starter doctor passes.
- Starter self-test, focused starter tests, and repo doctor pass.

## Release Boundary

This is `v0.1.0-rc.1`, a release candidate. It is not stable `v0.1.0`.

## Not Included

- No hosted service.
- No CLI product surface beyond helper scripts.
- No required Linear, GitHub, or Slack adapter.
- No first-class epic artifact support.
- No package registry publication.
- No private project history, run logs, event receipts, secrets, or personal paths in the starter archive.

## First Use

Extract the archive into a fresh project, then ask the agent to read `AGENTS.md` and `.ai-os/current.md`.
