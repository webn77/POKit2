# POKit Starter v0.2.0

## Summary

POKit Starter v0.2.0 adds PO decision-tracking surfaces, lifecycle card response standard, agent profile dispatcher, and sprint backlog/release-scope artifacts while keeping the natural-language entrypoint and `.ai-os/` source-of-truth principles unchanged.

This is a minor (backward-compatible) release. Existing v0.1.0 starters can adopt v0.2.0 features incrementally â€” none of the new fields are required.

## Migration from v0.1.0

v0.2.0 is backward compatible. Existing v0.1.0 starters can:

1. Extract `pokit-starter-v0.2.0.tar.gz` over their project.
2. Files in `preserve_never_overwrite` (POK-*.md, current.md, status-board.md, issue-index.md, memory/, events/) are not touched.
3. Updated files: `scripts/pokit-runner.mjs`, `scripts/pokit-doctor.mjs`, and `.ai-os/standards/*.md` refresh to v0.2.0.
4. Optional: adopt new fields (`agent_profile`, `depends_on`, `goal`, `ai_self_verify`) on new issues. Existing issues keep their shape.
