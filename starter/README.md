# POKit2

설치해주셔서 고맙습니다. POKit2는 거친 요청을 이슈 기반 작업으로 바꾸고, 상태를 저장소 안에 두며, 근거 없이는 "완료"를 막는 로컬 우선 AI 작업 하니스입니다.

## 시작하기

대화 도구(예: Claude Code)에서 시작 문구를 입력하세요.

```
포킷 시작
```

상태 카드가 뜨고, 다음에 무엇을 할지 안내합니다.
